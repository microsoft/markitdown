7. Conclusion
In this paper, we define the issue of individual diversity in Hu-
man Activity Recognition (HAR). The deep coupling between activity
features and identity features poses a challenge for the scalability of
state-of-the-art HAR methods to new users. Therefore, we propose a
deep learning framework called CUDSG, which is the first attempt to
achieve cross-user domain action recognition in a few-shot scenario.
First,inordertoprovidesufficientdata,alow-cost,training-freedata
augmentationschemeisproposed.Second,afeaturedecouplingmod-
uleandagenerationmodulearepresenttoseparatethegesturefeatures
and identity features and generate virtual samples of the target user.
Fig.18. Impactofreceiver. Finally, we evaluated the performance of the model on the public
dataset. The results show that CUDSG can effectively improve the
accuracyofexistingclassificationmethods.
OurworkprovidesanewsolutionforHARformulti-userscenarios.
However,therearestillsomelimitationstothecurrentwork,andwe
will try to solve them in the future. First, CUDSG is mainly focused
on virtual sample generation cross user domains. In the future, we
aim to extend CUDSG to cross other domains, such as using training
samples collected from users in certain orientations (or locations) to
generate virtual gesture samples for the target orientation. Second,
WhileCUDSGcangeneratesamplesunderunsupervisedconditions,it
stillrequiresaccesstothetargetdomaindataset.Inthefuture,weaim
togeneratesampleswithoutaccessingthetargetdomaindatasetatall.
It is a popular domain generalization (DG) problem in deep learning
at present. Finally, The effect of individual differences on perceptual
effects is packaged as identity features, while in reality the boundary
between individual differences and gesture features is indistinct, and
